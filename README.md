# Lakshmi Nair - Portfolio

Personal brand hub for Lakshmi Nair - Builder & Founder.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Design System

### Colors (Editorial Palette)
- **Navy:** #0A1128 (primary, headlines)
- **Cream:** #F7F3E9 (backgrounds)
- **Terracotta:** #E07A5F (accent)
- **Charcoal:** #2D3142 (body text)

### Typography
- **Headlines:** Playfair Display (serif)
- **Body:** Inter (sans-serif)
